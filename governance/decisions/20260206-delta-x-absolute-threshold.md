# 20260206-delta-x-absolute-threshold

## Context
ΔE 합산 점수가 낮더라도 심각한 외부 전가를 가릴 수 있다는 우려가 반복적으로 제기되었다.

## Decision
`ΔX >= +3`이면 자동 악 판정을 적용한다.

## Alternatives Considered
1. 가중치만 강화 (`2×ΔX`)하고 자동 규칙은 두지 않는다.
2. 임계값을 +4 또는 +5로 높인다.

## Consequences
- 장점: "내부 효율" 명목의 외부 피해 전가를 강하게 차단
- 단점: 경계값(+2~+3)에서 판정 논쟁이 집중될 수 있음
