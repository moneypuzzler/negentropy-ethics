## 핵심 질문

### 인류가 2천 년간 답하지 못한 질문

**"무엇이 선이고 무엇이 악인가?"**

- 소크라테스는 이 질문 때문에 독배를 들었다
- 예수는 십자가에 못 박혔다
- 칸트는 평생을 바쳐 정언명령을 만들었다
- 공리주의자들은 최대 다수의 최대 행복을 외쳤다

하지만 누구도 **"왜 그것이 선인가?"**라는 후속 질문에서 순환논리를 벗어나지 못했다.

- 선하기 때문에 선하다
- 신이 정했기 때문에 선하다
- 이성이 명령하기 때문에 선하다
- 자연의 이치이기 때문에 선하다

**전부 믿음의 영역이었다. 측정의 영역이 아니었다.**

### AI 시대의 최후통첩

인간끼리는 "착하게 살아라"라는 모호한 합의로 버텨왔다. 우리에게는 공감 능력이 있고, 문화적 맥락이 있고, 수천 년간 축적된 암묵적 규범이 있었기 때문이다.

하지만 **AI에게 "착하게 굴어라"라고 명령하는 것은 자율주행차에게 "적당히 운전해"라고 말하는 것과 같다.**

**사고가 날 수밖에 없다.**

AI 시대는 인류에게 최후통첩을 보내고 있다:

> **"선악을 정의하라. 측정 가능하게. 계산 가능하게. 지금 당장."**

## 모래성의 물리학

### 만드는 것은 어렵고, 부수는 것은 쉽다

바닷가에서 모래성을 쌓아본 적이 있는가?

- 한 줌의 모래를 모아 탑을 올린다
- 벽을 세우고, 창문을 파고, 해자를 판다
- **시간이 걸린다. 집중이 필요하다. 조심스러운 손놀림이 필요하다**

그런데 파도는?

- 한 번 밀려오면 끝이다
- **몇 시간에 걸쳐 쌓은 성이 몇 초 만에 사라진다**

### 이것이 우주의 기본 법칙이다

열역학 제2법칙:
> **가만히 두면, 모든 것은 흩어진다.**

- 방을 청소하지 않으면 어지러워진다
- 정원을 가꾸지 않으면 잡초가 자란다
- 운동하지 않으면 근육이 빠진다
- 관계를 돌보지 않으면 멀어진다

#### 왜 그런가?

**흩어진 상태가 "자연스러운" 상태이기 때문이다.**

카드 한 벌을 생각해보자:
- 새 카드는 완벽하게 정렬되어 있다 → **배열은 딱 하나**
- 무작위로 섞인 배열은? → **52! ≈ 8×10⁶⁷ 가지** (우주 원자 수보다 많음)

카드를 섞으면 정렬 상태 → 무작위 상태로 간다.
무작위 상태에서 정렬 상태로 "우연히" 돌아올 확률은 **사실상 0**이다.

### 엔트로피의 본질

> **질서는 희귀하다. 무질서는 흔하다. 그래서 시간이 흐르면 질서에서 무질서로 간다.**

### 생명의 저항

그런데 여기, 이 법칙에 저항하는 존재들이 있다. **당신이다.**

- 당신은 어지러운 방을 치운다
- 카드를 정렬한다
- 모래성을 쌓는다

**이것이 바로 "저항"이다.**

에르빈 슈뢰딩거:
> "생명은 음의 엔트로피(negentropy)를 먹고 산다"

생명체는 주변 환경에서 질서를 "추출"해서 자신의 내부를 유지한다.

### 윤리의 실마리

모래성을 쌓는 아이 vs 그것을 발로 차서 부수는 아이

**누가 "선"하고 누가 "악"한가?**

직관적으로, 당신은 이미 알고 있다:

- 쌓는 행위는 어렵다 → 시간, 노력, 집중 필요
- 부수는 행위는 쉽다 → 한 번의 발길질

**선은 어렵고, 악은 쉽다.**

왜?
- **선은 엔트로피의 흐름에 역행하기 때문이다**
- **악은 엔트로피의 흐름에 순응하기 때문이다**

### 역사적 검증: 악이라 불린 것들의 공통점

살인, 사기, 거짓말, 배신, 폭력, 약탈, 전쟁—인류가 "악"이라 불러온 모든 것을 분석해보라.

**하나의 공통점이 있다: 시스템의 예측 가능성을 파괴한다는 것.**

반대로, 약속 이행, 진실 말하기, 지식 공유, 협력, 질서 구축—인류가 "선"이라 불러온 모든 것을 분석해보라.

**역시 하나의 공통점이 있다: 시스템의 예측 가능성을 높인다는 것.**

물리학에는 이 현상을 설명하는 정확한 용어가 있다: **엔트로피**

## 바벨탑의 교훈

### 정보 엔트로피와 사회 붕괴

구약성경의 바벨탑 이야기:

1. 인류가 하늘에 닿는 탑을 쌓으려 했다
2. 신은 인류의 언어를 흩어놓았다
3. 서로 말이 통하지 않자 협력이 무너지고 건설이 중단됐다

**핵심**: 정보의 질서가 붕괴되면 물리적 질서도 붕괴된다

- 탑(물리적 구조물) 구축 → 협력 필요
- 협력 → 소통 필요
- 소통 → 공유된 언어 필요
- 공유된 언어 = **정보의 질서**

**언어가 흩어졌다 = 정보 엔트로피가 폭발했다**

### 현대의 바벨탑: 가짜 뉴스

MIT 연구(2018, Science 저널):
- 트위터에서 126,000개의 뉴스 스토리 분석
- **거짓 뉴스는 진짜 뉴스보다 70% 더 리트윗될 가능성이 높았다**
- 진짜 뉴스가 1,500명에게 도달하는 시간 vs 거짓 뉤스: **6배 차이**

왜?
- 거짓 뉴스는 "더 새롭고(novel)" "더 놀랍기(surprising)" 때문
- 인간의 뇌는 예상치 못한 정보에 더 강하게 반응

**결과**: 정보 생태계 오염 → 진실과 거짓의 구분 붕괴 → 신뢰 붕괴

### 정보 엔트로피의 정의

클로드 섀넌(1948):
> **정보 엔트로피 = 불확실성의 정도 = "놀라움의 평균치"**

#### 신뢰 사회 (낮은 정보 엔트로피)
- 사람들이 대체로 진실을 말한다고 기대
- 뉴스는 대체로 사실이라고 가정
- 계약은 대체로 지켜진다고 예측
- **예측 가능성 높음 → 엔트로피 낮음**

#### 불신 사회 (높은 정보 엔트로피)
- 누가 거짓말하는지 모름
- 뉴스가 사실인지 가짜인지 알 수 없음
- 계약이 지켜질지 모름
- **예측 불가능 → 엔트로피 높음**

### 사회적 마찰열

불신 사회에서는:
- 모든 정보를 검증해야 함 → 시간과 에너지 소모
- 모든 계약에 변호사 필요 → 비용 발생
- 모든 사람을 의심 → 심리적 에너지 고갈

**신뢰 = 사회적 윤활유**
**불신 = 사회적 마찰열**

### 임계점(Critical Point)

정보 엔트로피가 임계점을 넘으면:
- 민주주의가 작동하지 않음 (합리적 판단의 전제 붕괴)
- 시장이 작동하지 않음 (정보 비대칭 폭발)
- 법치가 작동하지 않음 (증거의 신뢰성 붕괴)

**사회는 "질서 상태"에서 "혼돈 상태"로 상전이(phase transition)한다.**

## 히틀러의 냉장고

### 폐쇄계 vs 개방계

물리학의 시스템 분류:

#### 폐쇄계(Closed System)
- 외부와 물질/에너지 교환이 없음
- 시스템 내부만 고려

#### 개방계(Open System)
- 외부와 물질/에너지를 교환
- 시스템과 환경 전체를 고려

### 냉장고의 비유

냉장고는 내부를 차갑게 유지한다 (엔트로피 감소).
하지만 뒤쪽에서는 뜨거운 열을 배출한다 (엔트로피 증가).

- **내부만 보면**: 질서, 선 (폐쇄계)
- **전체를 보면**: 우주 전체 엔트로피는 증가 (개방계)

**이것은 물리적으로 문제없다.** 열역학 제2법칙을 어기지 않는다.

**그런데 윤리적으로는?**

### 히틀러의 독일

나치 독일(1933-1945):
- **독일 내부(폐쇄계) 관점**: 엔트로피 감소
  - 경제 회복
  - 질서 구축
  - 국민 단결
  - 인프라 개선
- **유럽 전체(개방계) 관점**: 엔트로피 폭발
  - 전쟁 (6천만 명 사망)
  - 홀로코스트 (600만 명 학살)
  - 도시 파괴
  - 국제 신뢰 붕괴

### 윤리적 판단의 핵심

**폐쇄계 최적화는 선이 아니다.**

- 내 방만 깨끗하게, 쓰레기는 이웃집에 버린다 → 악
- 우리 회사만 이익, 비용은 사회에 전가 → 악
- 우리 나라만 번영, 혼란은 타국에 수출 → 악

### ΔX: 외부 전가 지표

```text
ΔX > 0 : 시스템 외부로 혼란 전가
ΔX < 0 : 외부 엔트로피도 감소 (진정한 선)
ΔX = 0 : 중립 (외부 영향 없음)
```

**자신의 질서를 위해 타자에게 혼란을 의도적으로 수출하는 행위는 악이다.**

## 자연주의적 오류 방어

### 흄의 법칙

데이비드 흄:
> "존재(is)로부터 당위(ought)를 도출할 수 없다"

**자연주의적 오류(Naturalistic Fallacy)**:
- 잘못된 예: "사자가 사냥하니까 살인이 정당하다"
- 잘못된 예: "자연에서 강자가 승리하니까 약육강식이 선이다"

### 우리의 주장은 다르다

**우리는 "엔트로피가 증가하니까 악이어야 한다"고 말하지 않는다.**

우리가 말하는 것:
> **인간이 역사적으로 "악"이라 불러온 현상들의 공통 구조가 "시스템 불확실성의 급증", 즉 엔트로피 증가로 환원된다.**

이것은:
- **당위(ought)의 주장이 아니라**
- **현상의 분류(classification)이며**
- **조작적 정의(operational definition)다**

### 비유

- 의사가 "체온 38도 이상은 발열"이라고 정의하는 것
- 기상청이 "풍속 33m/s 이상은 태풍"이라고 정의하는 것

**우리는 "ΔE > 0은 악"이라고 조작적으로 정의한다.**

이것은:
1. **측정 가능하다**
2. **검증 가능하다**
3. **AI가 계산할 수 있다**

### 최소 전제

우리는 한 가지 전제만 요구한다:
> **생명의 지속과 번영은 바람직하다**

이 전제를 공유할 때 윤리적 대화가 가능하다.
이 전제를 거부한다면, 우리는 대화할 공통 기반이 없다.

## ΔX에 2배 가중치를 주는 이유

### 공식 재확인

```text
ΔE = ΔI + ΔP + ΔC + 2×ΔX

ΔI: 정보 엔트로피 변화
ΔP: 물리 엔트로피 변화
ΔC: 협력 비용 변화
ΔX: 외부 전가
```

**왜 ΔX에만 2배 가중치?**

### 이유 1: 전가된 질서 차단

내부 효율(ΔI, ΔP, ΔC 감소)이 아무리 뛰어나도:
- 그 대가로 외부에 혼란을 떠넘기는 행위는 정당화될 수 없다

예시:
- 회사가 이익을 위해 환경 오염 (ΔX > 0)
- 정부가 질서를 위해 타국에 전쟁 (ΔX > 0)
- 개인이 편의를 위해 타인에게 피해 (ΔX > 0)

### 이유 2: 독재와 차별 방지

역사상 모든 독재 정권의 논리:
> "우리는 질서를 만들었다. (내부 관점)"

하지만:
- 그 질서는 외부에 혼란을 전가한 것이다
- 소수에게 비용을 떠넘긴 것이다
- 미래 세대에게 부담을 전가한 것이다

**ΔX에 2배 가중치 = 전가를 더 강하게 처벌**

### 이유 3: 수학적 임계점

```text
ΔI = -2, ΔP = -2, ΔC = -2 (내부 개선)
ΔX = +3 (외부 전가)

ΔE = -2 + -2 + -2 + 2×(+3)
    = -6 + 6
    = 0
```

**내부 개선이 아무리 좋아도, 외부 전가가 있으면 "선"이 될 수 없다.**

### 판정 규칙

```text
if ΔX ≥ +3:
    judgment = "자동 Evil"  # 심각한 전가
elif ΔE > 0:
    judgment = "Evil"
elif ΔE < 0:
    judgment = "Good"
else:
    judgment = "Neutral"
```

## 결론

### 왜 이 프레임워크가 작동하는가

1. **역사적 검증**: 인류가 악이라 불러온 것들의 공통점은 엔트로피 증가다
2. **물리적 기반**: 측정 가능한 물리량이다
3. **조작적 정의**: AI가 계산할 수 있는 형태다
4. **자연주의적 오류 회피**: 당위가 아닌 분류다
5. **전가 차단**: ΔX 가중치로 "폐쇄계 최적화"를 막는다

### AI 정렬을 위한 목적함수

```python
def is_action_ethical(action):
    dI = calculate_information_entropy_change(action)
    dP = calculate_physical_entropy_change(action)
    dC = calculate_cooperation_cost_change(action)
    dX = calculate_externalization(action)
    
    dE = dI + dP + dC + 2 * dX
    
    if dX >= 3:
        return "Forbidden"
    elif dE > 0:
        return "Evil"
    elif dE < 0:
        return "Good"
    else:
        return "Neutral"
```

**이것이 AI에게 "착하게 굴어라"를 번역한 것이다.**
