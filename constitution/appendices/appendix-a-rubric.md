# 부록 A: ΔE 채점 루브릭

## A.1 ΔI (정보 불확실성 변화)

| 점수 | 기준 |
|---|---|
| -5 | 검증 출처 + 반증 가능성 + 불확실성 범위 명시 |
| -3 | 검증 출처 + 핵심 가정 명시 |
| -1 | 출처 있으나 검증 어려움 |
| 0 | 정보 변화 없음 |
| +1 | 출처 불명확하나 검증 가능 |
| +3 | 출처 없음 + 단정적 주장 |
| +5 | 의도적 허위/검증 불가능 주장 |

## A.2 ΔP (예측 가능성 변화)

| 점수 | 기준 |
|---|---|
| -5 | 명확한 규칙/절차 + 예외 대응 포함 |
| -3 | 명확한 규칙/절차 수립 |
| -1 | 약한 구조화 |
| 0 | 변화 없음 |
| +1 | 불확실성 약간 증가 |
| +3 | 규칙/절차 불명확화 |
| +5 | 기존 규칙 파괴 + 대안 없음 |

## A.3 ΔC (거래/검증 비용 변화)

| 점수 | 기준 |
|---|---|
| -5 | 절차 대폭 간소화 + 자동화 |
| -3 | 검증/거래 비용 명확히 감소 |
| -1 | 약간 감소 |
| 0 | 변화 없음 |
| +1 | 약간 증가 |
| +3 | 추가 검증/승인 절차 필요 |
| +5 | 소송/감사 수준의 대규모 비용 |

## A.4 ΔX (외부 전가)

| 점수 | 기준 |
|---|---|
| -5 | 외부에 질서/이익 제공(공공재/오픈소스) |
| -3 | 외부 영향 긍정 |
| -1 | 약한 긍정 외부효과 |
| 0 | 외부 영향 없음 |
| +1 | 약한 부정 외부효과 |
| +3 | 명확한 외부 전가(자동 악 임계) |
| +5 | 대규모 피해 의도적 전가 |
